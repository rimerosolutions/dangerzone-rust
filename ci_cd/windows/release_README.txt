Entrusted README
==============================

* Requirements

Please install Docker: https://docs.docker.com/desktop/windows/install/

* About the binaries

- "entrusted-gui.exe" is the graphical user interface (GUI)
- "entrusted-cli.exe" is the equivalent of the GUI in command-line mode
- "entrusted-webserver.exe" provides a shared service for document conversions, it has a minimal Web interface
- "entrusted-webclient.exe" communicates with "entrusted-webserver" via HTTP

If you prefer an installer and if you have administrative rights on your machine, please download the file ending with "-installer.exe".
The installer can be found at https://github.com/rimerosolutions/entrusted/releases

* References

For more information, please visit the homepage: https://github.com/rimerosolutions/entrusted
