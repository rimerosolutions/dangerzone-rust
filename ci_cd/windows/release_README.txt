Dangerzone README
==============================

* Requirements

Please install Docker: https://docs.docker.com/desktop/windows/install/

* About the binaries

- "dangerzone-gui.exe" is the graphical user interface (GUI)
- "dangerzone-cli.exe" is the equivalent of the GUI in command-line mode
- "dangerzone-httpserver.exe" provides a shared service for document conversions, it has a minimal Web interface
- "dangerzone-httpclient.exe" communicates with "dangerzone-httpserver" via HTTP

If you prefer an installer and if you have administrative rights on your machine, please download the file ending with "-installer.exe".
The installer can be found at https://github.com/rimerosolutions/dangerzone-rust/releases

* References

For more information, please visit the homepage: https://github.com/rimerosolutions/dangerzone-rust
